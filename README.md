## Deploy to Vercel in 4 steps

### Step 1 — Create a GitHub account
Go to **github.com** → click Sign Up → use any email.

### Step 2 — Upload this project to GitHub
1. Once signed in, click the **+** button (top right) → **New repository**
2. Name it `babywatch`, leave everything else as default, click **Create repository**
3. On the next page, click **uploading an existing file**
4. Drag the entire contents of this `babywatch` folder into the upload area
5. Click **Commit changes**

### Step 3 — Create a Vercel account & deploy
1. Go to **vercel.com** → click Sign Up → choose **Continue with GitHub**
2. Click **Add New Project**
3. Find `babywatch` in the list → click **Import**
4. Leave all settings as they are → click **Deploy**
5. Wait about 60 seconds — done!

### Step 4 — Share with your family
Vercel gives you a link like `babywatch-xyz.vercel.app`.
- Send that link to your family by text or email
- On iPhone: open the link in Safari → tap the Share button → **Add to Home Screen**
- On Android: open in Chrome → tap the three dots → **Add to Home Screen**
It will appear as an app icon on their phone.

---

## Making changes later
Edit any file in GitHub (click the file → pencil icon to edit).
Vercel automatically re-deploys within 30 seconds. The link stays the same.

---

## Custom domain (optional)
In Vercel → your project → Settings → Domains → add any domain you own.
Free domains are available at **freenom.com** or you can buy one at **namecheap.com** (~$10/yr).
